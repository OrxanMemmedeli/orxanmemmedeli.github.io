Mən Məmmədəli Orxan Bəhram oğlu. Microsoft firmasının müasir proqramla dili olan C#(sharp) üzrə Junior Developer-əm.

Bakalavr təhsilimi AVTOMATİKA, Magistr təhsilimi isə İnformasiya Texnologiyaları (IT) üzrə almışam. Magistr dövründə Microsoft firmasının məhşur proqramlama dili olan C# biliklərini daha yaxşı mənimsəmək üçün proqramlama kursunda bilik və bacarıqlarımın artırılmasına çalışmışam. Növbəti mərhələdə müxtəlif platformalar ilə cari tarixə qədər yeni biliklər öyrənməyə davam edirəm. 
Əldə etdiyim biliklər üzrə Desktop App (windows), Web App (windows və Cross platform) olmaqla proqram təminatları hazırlamaq cəhtləri etmişəm. Günümüzün tələblərini nəzərə alaraq hazırlamağa üstünlük verdiyim tətbiqlər veb tətbiqlər olmaqla həm windows, həm də cross platrform olmaqla layihələr hazırlamışam və davam edirəm.

Mənimlə əlaqəli digər məlumatlara daha dəqiq formada səhifə üzərindən baxa, ehtiyac yarandığı təqdirdə əlaqə bölməsindən əlaqə saxlaya bilərsiniz.
